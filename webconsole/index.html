Web console (dev)

Point api-base at any service that implements POST …/api/cli with body { "line": "…" } and POST …/api/cli/complete for Tab completion, and POST …/api/cli/login with username + password or sessionToken (not in the line string). For local dev, Vite proxies /api to http://127.0.0.1:3042. In this monorepo, run the API from server/ (npm start after npm run build) or use docker compose -f server/docker-compose.yml up from the repo root.
